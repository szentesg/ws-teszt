# Easy Readable Date String Generator

## Requirements

- Symfony 7.2.x
- PHP 8.3.x
- PHPUnit

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/szentesg/ws-teszt.git
cd ws-teszt
```

### 2. Composer install

```bash
composer install
```

### 3. Browser index page: (/)

### 4. Run Unit tests

```bash
php vendor/bin/phpunit
```
